## 3. North-star màn HUD

HUD = **trang chủ / mission-control sống**. Theo 3-mức-zoom của #0 §3, HUD là mức **"Chòm sao standby"**: **lõi = AETHER** toả tia ra **roster** các đích/agent. Ở AETHER, **roster = sessions** (mỗi phiên = một agent đang/đã làm việc trong hạm đội).

- **Lõi (core orb)** = AETHER: hữu cơ/phát quang, **nhân = đám mây hạt** (particle nucleus), halo, **thở**; vital-state theo `$gatewayState`.
- **Node** = **một session**; "nụ" phát sáng có chiều sâu (không phải chấm dẹt); trạng thái:
  - `online` (vừa hoạt động) = **azure**;
  - `busy` (`isActive`) = **hổ phách `--ae-energy`** + đốm data chạy dọc tua;
  - `dormant` (cũ/nghỉ) = **viền nét đứt** mờ.
- **Link** = **tua cong "flow-field"** xoáy mềm (wind-swept), data chạy dọc; node rìa thêm **circuit-trace** (gấp khúc kiểu PCB "dock ra rìa").
- **Canvas: LIGHT** (kính trắng frosted + navy/azure + amber energy). Dark fork bất động (không polish đợt này).

## 5. Kiến trúc & luồng dữ liệu (tái dùng engine #0)

HUD **không tự render WebGL** và **không thêm Canvas**. Nó **compose một `GraphSpec`** từ data thật rồi đẩy vào shared store; `AetherCanvas` ở shell-root tự vẽ.

```
$agents (sessions snapshot)  ─┐
                              ├─► sessions-graph (logic thuần) ─► GraphSpec ─► setGraphSpec($graphSpec) ─► AetherCanvas (R3F #0) vẽ chòm sao
lifecycle-differ (diff)  ─────┘                                    ▲
                                                                   └─ per-node lifecycle phase (mitosis/flow/inhale/prune) cho view animate
$briefing  ───► 4 widget ambient (DOM, GlassSlab)
$gatewayState / $orbState ───► vital lõi + vital-sign top-bar (đã có #0)
```

- **Tách lõi-logic vs view** (bắt buộc để test jsdom + giữ prompt-cache):
  - **(a) Logic thuần (test jsdom):** `sessions-graph.ts` map `AgentSessionRow[]` → `GraphSpec` (cap top-N, `constellationLayout`, gán `NodeState`); `lifecycle-differ.ts` diff snapshot trước/sau → list **lifecycle events**.
  - **(b) View R3F mỏng:** `AetherCanvas`/`GraphView` #0 (mở rộng) đọc `GraphSpec` + events → render + animate. Guard-test mount/dispose.
- **Poller:** hook `use-agents-poll.ts` gọi `loadAgents()` mỗi ~5s (interval, pause khi `document.hidden`), cập nhật `$agents`; HUD subscribe `$agents` → recompute `GraphSpec`. (Read-only snapshot → prompt-cache safe.)

## 6. Vòng đời session = snapshot-diff (cache-safe)

`lifecycle-differ` so snapshot `$agents.sessions` lần trước với lần sau, phát **events** ở **mức session** → view biên đạo 6-verb #0:

| Sự kiện diff (coarse metadata) | Verb #0 | Hiệu ứng view |
|---|---|---|
| session id mới xuất hiện | reach + **mitosis** | nụ tách khỏi lõi (phân bào), tua flow-field mọc nối |
| `isActive` true (đang chạy) | **flow** | đốm `--ae-energy` chảy **ra** dọc tua (ambient, **không** phải token thật) |
| `messageCount` tăng / vừa kết thúc | **inhale** | đốm chảy **về lõi** |
| session hoàn tất (active→inactive) | **crystallize** | 1 ánh kết tinh nhỏ ở node (result modal đầy đủ = #3 Chat) |
| session biến mất / quá cũ (rớt top-N) | **prune** | node co/tan trở về lõi |

- **GraphSpec mở rộng (tối thiểu):** mỗi `NodeSpec` thêm hint vòng đời cho view nội suy enter/exit (vd `enter?: boolean`/`exit?: boolean` hoặc `phase`+`since`); chốt hình thức cụ thể ở plan. Layout giữ ổn định (deterministic, không RNG) để node không "nhảy" giữa các poll.
- **Cap & ordering:** top-N=12; ưu tiên `isActive`, rồi `lastActive` desc. **Rỗng** (`sessions.length===0`) → **skeleton**: lõi thở một mình + dòng gợi ý "Chưa có phiên — bắt đầu trò chuyện".
- **Tuyệt đối không** subscribe stream; mọi "data chạy" là **ambient decoration** suy từ `isActive`/`messageCount`, không phải token.

## 9. Tích hợp shell, tương tác & a11y

- **Bỏ `<CommandBar>` trong-màn** ([command-center.tsx:121](../../apps/desktop/src/aether/ui/screens/command-center.tsx#L121)) — đóng carry-over dual-⌘K #0. ⌘K dùng top-bar global.
- **Tokenize sạch** mọi hardcode màu → `--ae-*` (+ `--ae-energy` cho amber-đang-làm).
- **Tương tác:** click node (raycast) → mở session đó (`/:sessionId`); hover → tooltip nhãn.
- **a11y:** constellation là **enhancement** (raycast click); điều hướng chuẩn vẫn qua **nav-rail DOM**; 4 widget ambient là DOM thật, focusable, đọc được bằng screen-reader. Vital lõi không dựa vào chỉ màu (kèm nhãn).

## 10. Fallback (hard-rule #0)

`reduced-motion` **HOẶC** GPU-off (`getRemoteDisplayReason()`) **HOẶC** webgl-probe fail → **không mount Canvas** → `GraphFallback` ([fallback.tsx](../../apps/desktop/src/aether/ui/motion/graph/fallback.tsx)) render chòm sao **tĩnh** từ cùng `GraphSpec`; **4 widget ambient hiển thị bình thường**. Poller vẫn cập nhật trạng thái tĩnh (state node đổi màu) nhưng **không** chạy animation verb. → all-WebGL = đường chính, **vẫn phải** kèm đường fallback.

## 12. Testing

- **Gate mỗi slice:** `npx vitest run --environment jsdom src/aether && npx tsc -p . --noEmit` xanh.
- **Unit thuần (jsdom):**
  - `sessions-graph`: cap top-N=12 đúng thứ tự (active trước, lastActive desc); layout ổn định (không nhảy giữa poll); map `isActive`→busy, recent→online, cũ→dormant; rỗng→skeleton spec.
  - `lifecycle-differ`: mỗi loại diff (mới/active/messageCount↑/hoàn tất/biến mất) → đúng event; không phát event giả khi snapshot không đổi.
  - 4 widget render đúng từ `$briefing`/`$agents`; trạng thái hạm đội = `runningCount`.
  - HUD **không còn** render `CommandBar`; **hết** hardcode màu (assert dùng `--ae-*`); title "Trang chủ".
  - Click node → điều hướng `/:sessionId`.
- **Guard (WebGL, jsdom không chạy GL):** mount/unmount/dispose; `clearGraphSpec` khi unmount; không rò BufferGeometry.
- **Fallback gate:** reduced-motion/GPU-off/probe-fail → **không** Canvas, có `GraphFallback` + 4 widget.
- **Thủ công (máy thật):** Light đẹp; chòm sao flow-field "tự nhiên"; spawn/flow/inhale/prune chạy khi mở/đóng phiên thật; remote-display → fallback (không Canvas trắng); poll không spike CPU; reduced-motion → tĩnh.
- Mỗi bước TDD: đỏ → code → xanh → commit.

## 13. Rủi ro & giảm thiểu

- **Prompt-cache vỡ nếu lỡ subscribe stream** → chỉ ăn snapshot/`$gatewayState`; review chặn mọi import stream vào HUD.
- **Node "nhảy" giữa các poll** (layout không ổn định) → layout deterministic theo id/thứ tự, không RNG; cap ổn định.
- **Poll gây tải** → interval ~5s, pause khi `document.hidden`, tái dùng `loadAgents()` read-only; cân nhắc backoff khi lỗi.
- **Số session dao động 0↔nhiều** → cap top-N + skeleton khi rỗng; enter/exit có animation prune để không "biến mất đột ngột".
- **Rò three.js** (BufferGeometry) → dispose (carry-over #0) — bắt buộc vì HUD render bền.
- **Scope creep sang Chat/Brief** → ranh giới §14 cứng.
- **Drift token** → test assert `--ae-*`, không hardcode.

## 14. Ranh giới scope

1. HUD rebuild **Light · L2** (`final-hud-l2.html`): chòm sao flow-field B2 tràn nền + 4 widget ambient.
